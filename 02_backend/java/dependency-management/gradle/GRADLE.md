# GRADLE

### Ejecutar proyecto SpringBoot: 
```shell script
gradlew bootRun
```

### Construir archivo jar: 
```shell script
gradlew build 
```

### Ejecutar archivo .jar: 
```shell script
java -jar build/libs/service-name-0.1.0.jar
```

### Ejecutar pruebas unitarias: 
```shell script
gradlew test
```
