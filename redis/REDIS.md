# REDIS

▶️ **Encender servidor de redis**
```shell script
redis-server
```

----

▶️ **Encender cliente de redis** 
```shell script
redis-cli
```

----

▶️ **Mostrar hashes** 
```shell script
keys *
```

----

▶️ **Borrar cache** 
```shell script
flushall
```

----

▶️ **Obtener por llave** 
```shell script
HGETALL persona:01
```
