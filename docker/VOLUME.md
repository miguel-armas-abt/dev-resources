# Volúmenes
Para conocer un poco más sobre volumes y mapeo de puertos visitar el siguiente enlace: 
<https://youtu.be/GwnDA-oXShI>

### Mostrar todos los volúmenes:
```shell script
docker volume ls
```

### Eliminar un volumen:
```shell script
docker volume rm <volume-id>
```

### Eliminar todos los volúmenes:
```shell script
docker volume prune
```
