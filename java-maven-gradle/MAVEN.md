# Maven

### Ejecutar proyecto SpringBoot: 
```shell script
mvnw spring-boot:run
```

### Compilar
```shell script
mvnw clean package
```

### Ejecutar archivo .jar: 
```shell script
java -jar target/service-name-0.1.0.jar
```

### Ejecutar pruebas unitarias: 
```shell script
mvnw test
```
### Dependencies
-Djps.track.ap.dependencies=false
